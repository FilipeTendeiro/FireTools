What is FireTools?

Automated Wildfire Analysis from UAV Footage

FireTools is a web-based platform that enables firefighting teams to upload RGB video footage from UAVs or surveillance towers and receive automated fire and smoke analysis. No technical expertise is required to operate it.

The platform processes videos through two deep learning pipelines ‚Äî detection for real-time bounding boxes, and segmentation for pixel-level fire and smoke masks.

Both models were trained and evaluated across multiple wildfire datasets totalling 121,454 training images (detection) and 2,471 images (segmentation).

ForesFireDataset D-Fire FASDD_UAV FASDD Corsican Fire FireFront-Gestosa GP Fire HugoLima LK Smoke
FireDetect ¬∑ YOLOv8n

Real-Time Detection

YOLOv8n (nano, 3.2M parameters) performs real-time bounding-box detection of fire and smoke at 82.7 FPS with an F1-score of 0.603.

FireSeg ¬∑ Hybrid DeepLabv3+

Pixel-Level Segmentation

Hybrid DeepLabv3+ with YOLO-guided smoke enhancement (26.7M parameters) delivers pixel-level segmentation at 30.6 FPS with smoke IoU of 0.525.

Interface

Designed for Operational Use

Three screens cover the full workflow ‚Äî upload, detection review, and segmentation analysis.

Upload & Run Detection

Drop video files onto the upload area or browse to select them. Processing tasks appear in a queue showing real-time status (processing / completed).

Once complete, navigate directly to the Detection Gallery to explore results.

  • Supports batch video uploads
  • Live processing status per file
  • Instant navigation to results

Detection Gallery

Side-by-side view of original and annotated videos. A color-coded timeline marks fire (pink), smoke (purple), and no detection (blue) for every frame.

  • Clear Images / Clear Videos / Clear All buttons
  • Playback: play/pause, ¬±0.5s, ¬±5s, speed 0.25√ó‚Äì2√ó, loop
  • Timeline scrubbing with color-coded activity
  • Launch segmentation directly from any detection item

Segmentation Gallery

View pixel-level segmentation results with two display modes: overlay (mask blended on original video) and mask-only view.

  • Overlay and mask-only display modes
  • Same timeline and playback controls as detection gallery
  • Start Segmentation from any detection item

Performance

Benchmark Results

YOLOv8n vs Faster R-CNN

YOLOv8n runs at 82.7 FPS and outperforms Faster R-CNN on F1-score (0.603 vs 0.287). Faster R-CNN has higher mAP but around 80% false positives, which limits practical use in the field.

Architecture Precision Recall F1-Score mAP@0.5 FPS
YOLOv8n 0.675 0.553 0.603 0.590 82.7
Faster R-CNN 0.192 0.629 0.287 0.698 15.0

Hybrid DeepLabv3+ vs Baselines

The hybrid DeepLabv3+ with YOLO Smoke guidance achieves the best smoke IoU (0.525) while keeping fire accuracy (0.627), with an 85.1% smoke detection rate at 30.6 FPS.

Quick Start

Get Running in Minutes

Choose your platform and follow the steps. Then open http://localhost:8000.

1

Install dependencies

winget install --id Python.Python.3.11 -e
winget install --id Gyan.FFmpeg -e
2

Clone and enter the folder

git clone <repository-url>
cd FireTools_Windows
3

Create virtual environment

py -3.11 -m venv .venv
.venv\Scripts\Activate.ps1
4

Install PyTorch (GPU ‚Äî CUDA 12.1)

python -m pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
5

Install remaining dependencies

python -m pip install -r requirements.txt
6

Run the app

cd scripts
py -3.11 app.py
